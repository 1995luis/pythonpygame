# Juego de Buscaminas en Consola

Este es un juego de **Buscaminas** realizado en Python utilizando la librer√≠a **pygame** para los efectos de sonido. El juego se ejecuta en la consola y permite al jugador intentar descubrir todas las celdas sin activar una mina. Si el jugador destapa una celda con una mina, pierde el juego. Si descubre todas las celdas sin activar ninguna mina, gana.

## Requisitos

- Python 3.x
- Pygame
